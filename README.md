ImpactVision

Decision Impact Prediction System

Impact Vision analyzes a person’s daily habits, lifestyle choices, or decisions and predicts their long-term impact on health, finances, and career.
Instead of focusing on a single domain, the system uses data-driven simulation models to evaluate multiple life dimensions and provide actionable insights for informed decision-making.

Features

Input daily habits or decisions in natural language

Predict long-term impact on health, financial stability, and career growth

Generate a comprehensive Impact Score and risk level

Provide future projections across multiple time horizons

Deliver clear explanations and personalized recommendations

Maintain history of past simulations and reports (optional dashboard)

How it Works (Pipeline)

User Input (habit / decision)
→ Preprocessing (text normalization and parameter extraction)
→ Behavior Classification (health / finance / career categories)
→ Impact Simulation Engine (domain-specific models)
→ Risk Scoring Model (combined weighted analysis)
→ Projection Generator (time-based forecasts)
→ Recommendation Engine (actionable guidance)
→ Dashboard Output

Tech Stack

Frontend: Next.js (Firebase Studio)

Backend/API: Python (FastAPI) or Firebase Cloud Functions

Database: Firebase Firestore

Authentication: Firebase Auth (optional)

Deployment: Firebase Hosting
